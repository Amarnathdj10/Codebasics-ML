A beginner-friendly machine learning repository containing hands-on implementations of core ML concepts using Python and scikit-learn.
This repo is focused on learning by coding, with clear examples, simple datasets, and well-structured notebooks/scripts.

What This Repository Covers:

This repository includes practical implementations of:
1)Data loading and exploration
2)Feature scaling (MinMaxScaler, StandardScaler)
3)Machine Learning algorithms
4)Naive Bayes (GaussianNB, MultinomialNB)
5)Train-test split
6)Pipelines in scikit-learn
7)Classic datasets like the Wine, Iris, and Digits dataset
8)Model training, evaluation, and accuracy scoring

Tech Stack:

Language: Python
Libraries: pandas, numpy, scikit-learn, matplotlib (where applicable)
